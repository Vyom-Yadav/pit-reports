43

44 
 * </p>
45 
 * <p>
46 
 * Rationale:
47 
 * Sometimes there are legitimate reasons for violating a check. When
48 
 * this is a matter of the code in question and not personal
49 
 * preference, the best place to override the policy is in the code
50

51

52

53

54

56

60

62
